Kensaku

This app was built using Create React App that calls giphy API. Users can search for, delete and select gifs.
I created Kensaku to garner more experience with React JS and with calling an API. I added on delete buttons as an extra feature so users could delete the gifs they did not like.



ReactJS, JavaScript ES6, Giphy API, HTML5, Sass, Heroku See it live: http://kensaku-gif.herokuapp.com.
One of the UI challenges I had was making sure the app design was responsive in desktop and mobile format. I used mixins and media queries to achieve this. I also adjusted the CSS so it was compatible for safari and mozilla browsers.
