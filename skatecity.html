Skate City

JavaScript, Ruby on Rails, HTML5, CSS3, WeChat

A basic todo list created with vanilla JS and minimalist design.
Users can discover, create and share skatespots with their WeChat contacts.


I created this simple app to consolidate my knowledge of JavaScript. I had also created a toggleAll function which would have toggled all todos completed. However, when it came to the UI design, I chose to leave this feature out as I preferred a clean and minimal input bar.

I had experimented with colour for the icons, however I wanted the todo list to look like a real notebook so I ended up sticking with a monochromatic colour scheme. I also added a bottom box shadow to the todo list container to resemble stacked paper.
