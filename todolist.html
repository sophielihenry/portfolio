Todo List

JavaScript ES6, HTML5, CSS3, Heroku
See it live:
http://daily-todo-list.herokuapp.com/index.html

A simple todo list created with vanilla JS and minimalist design. Users can add, delete, toggle and edit todos.

I created this simple app to practise and solidify my knowledge of JavaScript. I had also created a toggleAll function which would have toggled all todos completed. However, when it came to the UI design, I preferred I chose to leave this feature out as I preferred a clean and simple input bar over the todo list which I styled to look like a notebook.
