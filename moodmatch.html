Mood Match

Ruby on Rails, HTML5, CSS3, Bootstrap 4, Heroku See it live: http://moodmatch.herokuapp.com. Mood Match is an app built on Ruby on Rails that enables users to create and edit custom made cocktails.

This was my first time using Rails seed files to populate my database with cocktail ingredients. This also gave me experience working with JSON. One of the UI challenges I encountered was attempting to make the video background compatible for mobile view. In the end, I decided to create a static gradient background for mobile, as the load time was too slow.
